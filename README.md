        This is a ready-to-upload repository for a static Next.js site that shows upcoming football fixtures (mock data).

## How to run locally

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

3. Open http://localhost:3000

## Deploy to Vercel

1. Create a new GitHub repo and push this project.
2. In Vercel, import the GitHub repo and deploy.

This repository uses mock data in `lib/api.ts` so it works without any API keys.
